# DNA Replicável — Como usar

Esta pasta contém o **DNA genérico** extraído do projeto Drope, pronto pra
você plantar em qualquer projeto novo (Um Clique, Pet Shop, Sorveteria, o
que vier).

São **3 arquivos**, cada um com função clara:

| Arquivo | Função |
|---|---|
| `01-CONSTITUICAO-IA-SERVO.md` | A constituição. 10 artigos. Princípios fundamentais da IA-servo. Cópia direta. |
| `02-FRAMEWORK-MINISTERIOS.md` | O método de definir e operar os ministérios da IA. Cada projeto novo preenche a tabela em branco. |
| `03-METODOLOGIA-NOVO-PROJETO.md` | Passo-a-passo de como pegar um projeto novo e plantar o DNA do zero. |

---

## Como usar quando começar um projeto novo

1. **Copia esta pasta `_dna-replicavel`** inteira pra dentro do projeto novo.
2. Renomeia pra `_dna` (ou deixa o nome assim, tanto faz).
3. Abre o `03-METODOLOGIA-NOVO-PROJETO.md` e segue os 5 passos.
4. Quando terminar os 5 passos, você tem:
   - A Constituição adaptada pro projeto.
   - Os ministérios mapeados pro setor.
   - O motor desmontado.
   - Os princípios revalidados.

---

## O que muda de projeto pra projeto

- **NÃO muda:** os 6 princípios, a constituição, o método.
- **Muda:** os ministérios específicos do setor, o tom de voz, a paleta,
  os números reais.
